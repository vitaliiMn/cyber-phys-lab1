# Лабораторная работа №1 — Обнаружение объектов (Fire Detection)

## Описание задачи
Автоматическое обнаружение очагов возгорания на видеопотоке с камер наблюдения для систем раннего оповещения.

## Датасет
**Набор данных:** [Fire-detection (Kaggle)](https://www.kaggle.com/datasets/metinmekiabullrahman/fire-detection)

**Характеристики:**
- ~754 изображения с разметкой bounding boxes
- 1 класс: `fire`
- Формат: YOLO (отдельные папки images/labels для train/valid/test)
- Уникальный специализированный датасет (не COCO/Pascal VOC)

**Обоснование выбора:**
- Прямая привязка к практической задаче мониторинга пожарной безопасности
- Высокий визуальный контраст объектов (огонь хорошо различим)
- Разнообразие сценариев (домашние пожары, промышленные, открытое пламя)

## Метрики качества
| Метрика | Обоснование |
|---------|-------------|
| **mAP@0.5** | Mean Average Precision при IoU=0.5. Основной порог срабатывания тревоги — достаточно для фиксации факта наличия огня |
| **mAP@0.5:0.95** | Средний mAP при IoU от 0.5 до 0.95. Оценивает точность локализации bounding box для интеграции с системами пожаротушения |
| **Precision** | Доля правильных срабатываний. Минимизирует ложные тревоги (тени, блики, оранжевые объекты) |
| **Recall** | Доля найденных объектов. Минимизирует пропуски реальных очагов — критично для безопасности |
